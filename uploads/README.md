# Carpeta para almacenar archivos PDF
Este directorio contiene los archivos PDF que se visualizan en el flipbook.

## Instrucciones

1. Descarga tu archivo PDF
2. Nómbralo como `boletin.pdf`
3. Colócalo en esta carpeta
4. El aplicación cargará automáticamente el PDF

## Ejemplo
```
uploads/
└── boletin.pdf  ← Tu archivo PDF aquí
```

Si el PDF no se carga desde OneDrive, el sistema buscará el archivo en esta carpeta automáticamente.
